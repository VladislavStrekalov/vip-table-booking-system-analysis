# Нефункциональные требования

## Производительность

### NFR-1

Время отклика системы не должно превышать 3 секунд.

### NFR-2

Система должна поддерживать одновременную работу не менее 50 пользователей.

## Надёжность

### NFR-3

Система должна предотвращать создание двух бронирований одного столика на пересекающееся время.

## Доступность

### NFR-4

Система должна быть доступна пользователям не менее 99% времени.

## Безопасность

### NFR-5

Персональные данные клиентов должны храниться в защищённом виде.

## Удобство использования

### NFR-6

Пользователь должен иметь возможность создать бронирование не более чем за 5 действий.
