# Sequence Diagram

## Создание бронирования

### Описание

Диаграмма последовательности описывает взаимодействие между клиентом, системой бронирования и базой данных при создании бронирования VIP-столика.

Основной сценарий:

1. Клиент отправляет запрос на создание бронирования.
2. Система проверяет доступность выбранного столика.
3. База данных возвращает информацию о доступности.
4. Система сохраняет новое бронирование.
5. База данных подтверждает сохранение.
6. Система отправляет клиенту подтверждение бронирования.
